# Random-Forest
An implementation of Random Forest algorithm from scratch using Python.
## What it does
- Loads the built-in Digits dataset from sklearn (1797 images)
- Trains a Random Forest model with 20 decision trees
- Achieves ~97% accuracy on test data
- Visualizes results using a Confusion Matrix heatmap

## Libraries Used
- `pandas` - data manipulation
- `scikit-learn` - machine learning
- `matplotlib` - plotting
- `seaborn` - heatmap visualization
